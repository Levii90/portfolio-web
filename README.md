# 0xp1et Lab

Portfolio website untuk menampilkan hasil coding web Hafid Hidayat dengan tema dark cyber minimalist.

## Fitur

- Landing page dengan hero section, animasi, dan project carousel
- Dashboard portfolio dengan filter project
- Halaman detail project terpisah
- About section dan Contact section
- Responsive untuk desktop, tablet, dan mobile
- Routing menggunakan React Router DOM
- Animasi menggunakan Framer Motion
- Styling dengan Tailwind CSS

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- React Router DOM
- Framer Motion
- Lucide React

## Struktur

- `src/components`: komponen UI
- `src/data/projects.ts`: data project statis
- `src/pages`: halaman routing
- `src/types`: tipe TypeScript

## Menjalankan projek

1. Install dependencies

```bash
npm install
```

2. Jalankan development server

```bash
npm run dev
```

3. Build untuk produksi

```bash
npm run build
```

## Deploy

Website ini bisa dideploy ke Vercel secara langsung.

Pastikan `vercel.json` ada untuk support React Router.
